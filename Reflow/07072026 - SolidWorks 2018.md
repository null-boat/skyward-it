John Fulkerson responded to Windows 11 update offer to concerns about compatibility between SolidWorks 2018 and Windows 11.

From reading forums, most users reported that SolidWorks 2018 still works on Windows 11, but there have been some reports of minor issues:

- https://www.facebook.com/groups/2405941391/posts/10162998801241392/
- https://www.reddit.com/r/SolidWorks/comments/1gvk349/needs_help_i_need_to_install_solidworks_2018/

From [solidpeople.co.uk](https://solidpeople.co.uk/solidworks-hardware-requirements/?srsltid=AfmBOoqmlaNzxP2uLEjdK98QJWyDAw9lHWtTp5Mdfwswtt0kJ7tUcAVJ) :
"SOLIDWORKS 2022 or newer is required to support Windows 11 along with a PC with ==TPM 2.0== security support."

AFAIK backwards compatibility is not an issue, however forward compatibility is (maybe with the exception of SolidWorks 2018 → 2019)
